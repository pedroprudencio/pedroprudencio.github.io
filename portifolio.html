Pokedéx

A Pokédex é uma aplicação web desenvolvida com React e TypeScript. É uma ferramenta interativa que permite aos usuários explorar informações detalhadas sobre diferentes Pokémon, filtrar por tipo e marcar seus Pokémon favoritos.

Ichoveu

O projeto iChoveu é uma aplicação web que fornece informações meteorológicas e previsões do tempo para diferentes cidades. Desenvolvido com HTML, CSS e JavaScript, utiliza a biblioteca Fetch para realizar requisições à API WeatherAPI

Senha Segura

O Gerenciador de Senhas é uma aplicação web desenvolvida utilizando a biblioteca React. Seu principal propósito é permitir o cadastro seguro de informações sensíveis. Desenvolvido com TypeScript, Testing Library, UUID e CSS.
